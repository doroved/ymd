# Yandex Music Downloader — История изменений

> [!WARNING]
> Обязательно подпишитесь на наш Telegram-канал [@ymdownloader](https://t.me/ymdownloader), чтобы не потерять доступ к расширению, если оно вдруг пропадёт из магазина расширений!

## Версия 0.2.2 — 2026-06-11

### ✨ Новое

- **Кнопка скачивания в плеере** — В нижнюю панель плеера добавлена кнопка скачивания текущего трека.

### 🐛 Исправление

- **Правильные имена файлов в Chrome** — Исправлена ситуация, при которой другие расширения могли сбрасывать имя скачиваемого файла на UUID из blob URL.

### ⬆️ Улучшение

- **Обновлённые иконки расширения.**

## Версия 0.2.1 — 2026-06-09

### 🐛 Исправление

- **Корректное скачивание в 192 kbps** — Исправлена ошибка, при которой выбор качества 192 kbps в настройках всё равно скачивал треки в 320 kbps. Теперь выбранный битрейт корректно передаётся в API Яндекс.Музыки.

## Версия 0.2.0 — 2026-06-08

### ✨ Новое

- **Скачивание в FLAC (Lossless)** — Добавлена поддержка скачивания треков в формате FLAC без потерь качества. Для использования необходима подписка Яндекс Плюс. Если FLAC недоступен — автоматический fallback на MP3.
- **Выбор формата в настройках** — В настройках расширения теперь можно выбрать формат скачивания: MP3 (192/320 kbps) или FLAC (lossless). По умолчанию — MP3.
- **Метаданные во FLAC** — FLAC-файлы теперь содержат Vorbis Comments (название, исполнитель, альбом, год, жанр, номер трека) и встроенную обложку в формате PICTURE block.

### ⬆️ Улучшение

- **Визуальный прогресс скачивания** — Во время скачивания трека кнопка теперь отображает заполняющийся круг прогресса вместо спиннера. Для массового скачивания показывается прогресс каждого трека.
- **Модульная архитектура** — Код расширения полностью переписан с монолита на модульную структуру. Это повышает стабильность, упрощает поддержку и добавление новых функций.
