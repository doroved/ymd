# Yandex Music Downloader (YMD)

Расширение для браузера, которое позволяет скачивать музыку, подкасты и аудиокниги с Яндекс.Музыки — быстро, удобно и бесплатно.

---

## Возможности

### 🎵 Скачивание треков
Рядом с каждым треком на сайте Яндекс.Музыки появляется аккуратная кнопка скачивания. Нажали — трек уже на вашем компьютере.

### 📂 Скачивание альбомов, плейлистов, подкастов и аудиокниг
На странице любого альбома, плейлиста, подкаста или аудиокниги появляется кнопка **«Скачать всё»**. Одним нажатием скачиваются все треки с наглядным отображением прогресса. Загрузку можно остановить в любой момент.

### 🔊 Поддержка FLAC (Lossless)
Теперь можно скачивать треки в формате **FLAC без потерь качества**. Если FLAC недоступен — автоматический fallback на MP3.

### ⭕ Визуальный прогресс скачивания
Во время скачивания кнопка отображает заполняющийся круг прогресса. Для массового скачивания показывается прогресс каждого трека.

### 🏷 Теги и обложки
Каждый скачанный файл автоматически содержит всю информацию о треке: название, исполнителя, альбом, год, жанр, номер трека и обложку. Ваша медиатека всегда будет в порядке.

### ⚙️ Гибкие настройки
- **Формат аудио** — MP3 (по умолчанию) или FLAC (lossless)
- **Качество MP3** — высокое (320 kbps) или стандартное (192 kbps)
- **Обложки** — выбор разрешения от 400×400 до 1000×1000
- **Метаданные** — ID3-теги для MP3 и Vorbis Comments + PICTURE block для FLAC
- **Нумерация треков** — сохранение порядка треков в названии файла
- **Сортировка по папкам** — автоматическое распределение файлов по папкам (альбомы, плейлисты и т.д.)

## Установка

### Через Chrome Web Store (рекомендуется)

Установите расширение напрямую из магазина:

👉 [**Yandex Music Downloader в Chrome Web Store**](https://chromewebstore.google.com/detail/hemhffhlpdejhmbkapjjaciecddjehdm)

### Ручная установка (для разработчиков)

1. Скачайте или клонируйте этот репозиторий.
2. Откройте браузер (Chrome, Яндекс.Браузер, Edge, Opera) и перейдите на страницу `chrome://extensions/`.
3. Включите **«Режим разработчика»** в правом верхнем углу.
4. Нажмите **«Загрузить распакованное расширение»** и выберите папку `ymd`.
5. Откройте [Яндекс.Музыку](https://music.yandex.ru) — готово!
