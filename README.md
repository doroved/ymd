# Yandex Music Downloader (YMD)

Легковесное, невероятно быстрое и безопасное браузерное расширение для скачивания треков, плейлистов и альбомов с Яндекс.Музыки в высоком качестве (320 kbps) с автоматическим вшиванием ID3-тегов и обложек прямо в браузере.

---

## Основные возможности

### 1. Скачивание одиночных треков
* Рядом с каждым треком в интерфейсе Яндекс.Музыки появляется лаконичная желтая круглая кнопка скачивания.
* Скачивание осуществляется через **безопасную очередь** с ограничением до 7 одновременных потоков. Это предотвращает падение вкладок браузера из-за перегрузки оперативной памяти и защищает пользователя от блокировок/капчи со стороны серверов Яндекса.

### 2. Мгновенное скачивание альбомов и плейлистов
* В шапках страниц альбомов и плейлистов слева от кнопки «Слушать» добавляется кнопка «Скачать всё» (без лишних отступов, идеально встроенная в интерфейс).
* Использует современный **REST API подход**: расширение делает быстрые прямые запросы к официальным эндпоинтам Яндекс.Музыки (с параметром `richTracks=true`), мгновенно получая полные метаданные всех треков без использования медленных скрытых iframe, циклов ожидания или скрапинга DOM.
* Скачивание треков из плейлистов идет строго последовательно с безопасным интервалом в **1000мс** между началами загрузок, что полностью исключает риск срабатывания DDoS-защиты Яндекса.
* Кнопка динамически отображает красивый статус-бар прогресса (например, `12/45`) с анимацией загрузки, а по завершении показывает галочку «Готово!».

### 3. Автоматическое вшивание ID3-тегов и обложек
* Все метаданные вшиваются прямо в MP3-файл «на лету» в оперативной памяти браузера перед скачиванием на диск (с помощью библиотеки `browserId3Writer`).
* В файл внедряются:
  * Название трека и все исполнители
  * Альбом и год его выпуска
  * Порядковый номер трека (позиция) и общее количество треков в альбоме
  * Музыкальный жанр трека
  * Издатель (лейбл)
  * Обложка альбома (в выбранном в настройках разрешении)

### 4. Удобная страница настроек
У расширения есть стильная панель настроек в темных тонах, открывающаяся в отдельной вкладке:
* **Качество аудио**: Выбор между HQ (высокое качество, 320 kbps) и LQ (стандартное качество, 128 kbps).
* **ID3-теги**: Возможность включить или выключить автоматическое вшивание метаданных.
* **Сохранение в папки**: Автоматическое создание подпапок для скачанных файлов (например, сохранение всех треков в папку с названием альбома или исполнителя).
* **Позиция трека**: Возможность добавлять порядковый номер трека в название файла для сохранения оригинальной структуры плейлиста.
* **Размер обложки**: Выбор разрешения скачиваемой обложки (от 100x100 до 1000x1000).

## Архитектура и структура проекта

* `manifest.json` — Манифест расширения стандарта Manifest V3, оптимизированный под минимальные разрешения (`storage`, `downloads`).
* `sw.js` — Фоновый сервис-воркер (Service Worker) расширения, отвечающий за проксирование запросов к API Яндекс.Музыки в обход CORS и инициализацию скачивания файлов на диск.
* `content/` — Скрипты и стили страницы:
  * `content.js` — Основной скрипт: сканирует интерфейс, внедряет кнопки, взаимодействует с API Яндекс.Музыки, формирует очередь загрузки и вшивает теги ID3.
  * `tag.js` — Библиотека `browserId3Writer` для работы с бинарной структурой MP3 и записи тегов.
  * `content.css` — Стили кнопок скачивания, анимаций спиннера и интерактивных состояний прогресс-бара.
* `options/` — Страница настроек (HTML, CSS, JS) с современным адаптивным интерфейсом.
* `images/` — Иконки расширения.

## Установка

1. Скачайте файлы расширения или клонируйте репозиторий на компьютер.
2. Откройте любой браузер на базе Chromium (Google Chrome, Яндекс.Браузер, Microsoft Edge, Opera) и перейдите на страницу расширений (`chrome://extensions/`).
3. Включите **«Режим разработчика»** (Developer mode) in правом верхнем углу.
4. Нажмите кнопку **«Загрузить распакованное расширение»** (Load unpacked) в левом верхнем углу и выберите папку `ymd`.
5. Перейдите на [Яндекс.Музыку](https://music.yandex.ru) и наслаждайтесь скачиванием любимых треков!
